Propiedades margin, padding y border

Estas tres propiedades son fundamentales para controlar el espacio y la presentación visual de los elementos HTML. Son parte del modelo de caja de CSS y ofrecen un gran control sobre el diseño.

Margin (Margen)

El margin crea espacio alrededor del elemento, fuera de cualquier borde definido. Es transparente y no tiene color de fondo.

Propiedades individuales de margin

margin-top: 10px;
margin-right: 15px;
margin-bottom: 20px;
margin-left: 25px;

Notación abreviada (shorthand)

/* Todos los lados iguales */
margin: 10px;

/* Vertical (arriba/abajo) | Horizontal (izquierda/derecha) */
margin: 10px 20px;

/* Arriba | Horizontal (izquierda/derecha) | Abajo */
margin: 10px 20px 30px;

/* Arriba | Derecha | Abajo | Izquierda (sentido del reloj) */
margin: 10px 20px 30px 40px;

Casos especiales de margin

  • margin: 0 auto; - Centra horizontalmente un elemento de bloque con ancho definido
  • margin: auto; - Útil en flexbox y grid para alineación
  • margin-top: -20px; - Los márgenes negativos pueden superponer elementos
  • Los márgenes verticales adyacentes pueden colapsar (se fusionan)

Padding (Relleno)

El padding crea espacio alrededor del contenido de un elemento, dentro de cualquier borde definido. Hereda el color de fondo del elemento.

Propiedades individuales de padding

padding-top: 10px;
padding-right: 15px;
padding-bottom: 20px;
padding-left: 25px;

Notación abreviada (shorthand)

/* Todos los lados iguales */
padding: 10px;

/* Vertical (arriba/abajo) | Horizontal (izquierda/derecha) */
padding: 10px 20px;

/* Arriba | Horizontal (izquierda/derecha) | Abajo */
padding: 10px 20px 30px;

/* Arriba | Derecha | Abajo | Izquierda (sentido del reloj) */
padding: 10px 20px 30px 40px;

A diferencia de los márgenes, el padding:

  • No puede tener valores negativos
  • No colapsa con otros paddings
  • Está incluido en el área donde se muestra el fondo del elemento
  • Aumenta el tamaño total del elemento (a menos que uses box-sizing: border-box)

Border (Borde)

El border crea una línea alrededor del elemento, entre el padding y el margin. Tiene tres aspectos: ancho, estilo y color.

Propiedades individuales de border

/* Ancho del borde */
border-width: 1px;
border-top-width: 1px;
border-right-width: 2px;
border-bottom-width: 3px;
border-left-width: 4px;

/* Estilo del borde */
border-style: solid;
border-top-style: solid;
border-right-style: dashed;
border-bottom-style: dotted;
border-left-style: double;

/* Color del borde */
border-color: black;
border-top-color: red;
border-right-color: green;
border-bottom-color: blue;
border-left-color: yellow;

Notación abreviada (shorthand)

/* Todo en uno */
border: 1px solid black;

/* Por lado */
border-top: 1px solid red;
border-right: 2px dashed green;
border-bottom: 3px dotted blue;
border-left: 4px double yellow;

Estilos de border

CSS ofrece varios estilos de borde:

  • solid - Línea sólida
  • dashed - Línea discontinua
  • dotted - Línea de puntos
  • double - Línea doble
  • groove - Efecto 3D hundido
  • ridge - Efecto 3D elevado
  • inset - Efecto 3D presionado
  • outset - Efecto 3D saliente
  • none - Sin borde
  • hidden - Similar a none, pero afecta a la resolución de conflictos en tablas

La propiedad border-radius permite crear esquinas redondeadas. Si se establece en 50%, puede crear círculos (cuando el elemento es cuadrado) o elipses.

Comparación entre margin, padding y border

Propiedad Posición Color/Fondo Valores negativos Colapso
Margin Fuera del borde Siempre transparente Permitidos Pueden colapsar verticalmente
Border Entre margin y padding Color definido No permitidos No colapsan
Padding Dentro del borde, alrededor del contenido Hereda el color de fondo del elemento No permitidos No colapsan

Consejos prácticos

  • Usa margin para controlar el espacio entre elementos
  • Usa padding para controlar el espacio alrededor del contenido
  • Para centrar un elemento de bloque horizontalmente, usa margin: 0 auto; (debe tener un ancho definido)
  • Recuerda que los márgenes verticales colapsan entre elementos adyacentes
  • El padding aumenta el tamaño total de un elemento, a menos que uses box-sizing: border-box;
  • Los bordes también aumentan el tamaño total de un elemento
  • Para crear formas interesantes, experimenta con la propiedad border-radius

Estas tres propiedades son esenciales para el diseño en CSS. Dominarlas te permitirá controlar con precisión el espacio y la apariencia de tus elementos HTML.
