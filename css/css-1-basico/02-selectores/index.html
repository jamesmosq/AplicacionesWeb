Selectores básicos en CSS

Los selectores CSS te permiten apuntar a elementos HTML específicos para aplicarles estilos. Aprender a usar correctamente los selectores es fundamental para el diseño web.

Tipos de selectores básicos

1. Selector de elemento

Selecciona todos los elementos de un tipo específico.

p {
    color: blue;
}

Este es un párrafo afectado por un selector de elemento.

Este es otro párrafo afectado por el mismo selector.

2. Selector de clase

Selecciona todos los elementos que tienen un atributo de clase específico. Se usa un punto . seguido del nombre de la clase.

.destacado {
    font-weight: bold;
    color: #3498db;
}

Este es un párrafo normal.

Este párrafo tiene la clase "destacado".

Este div también tiene la clase "destacado".

Un elemento puede tener múltiples clases separadas por espacios: <p class="destacado grande">

3. Selector de ID

Selecciona un elemento con un ID específico. Un ID debe ser único en la página. Se usa el símbolo # seguido del nombre del ID.

#unico {
    background-color: #e8f4f8;
    padding: 10px;
    border-radius: 5px;
}

Este es un párrafo normal.

Este párrafo tiene el ID "unico" y solo puede existir uno en la página.

Los IDs tienen mayor especificidad que las clases o los selectores de elemento, lo que significa que sus estilos tienen prioridad.

4. Selector universal

Selecciona todos los elementos. Se representa con un asterisco *.

* {
    margin: 0;
    padding: 0;
}

El selector universal debe usarse con precaución, ya que afecta a todos los elementos y puede tener un impacto en el rendimiento si se aplica con propiedades costosas.

Modifica los siguientes selectores en la sección <style> del documento:

.practica-selectores p {
    /* Añade estilos para todos los párrafos dentro de .practica-selectores */
}

.practica-selectores .importante {
    /* Añade estilos para elementos con clase "importante" */
}

.practica-selectores #especial {
    /* Añade estilos para el elemento con ID "especial" */
}

Intenta aplicar diferentes propiedades como:

  • color
  • background-color
  • font-size
  • font-weight
  • padding
  • border
  • text-align

Comparativa de selectores

Tipo de selector Sintaxis Uso recomendado
Elemento elemento { } Estilos generales para un tipo de elemento
Clase .nombre-clase { } Estilos reutilizables en múltiples elementos
ID #nombre-id { } Elementos únicos con estilos específicos
Universal * { } Resetear estilos o aplicar reglas globales

Consejos finales

  • Usa selectores de elemento para estilos básicos y generales
  • Usa clases para la mayoría de los estilos reutilizables
  • Usa IDs con moderación, solo para elementos verdaderamente únicos
  • Usa el selector universal con precaución
  • La especificidad importa: ID > Clase > Elemento > Universal
