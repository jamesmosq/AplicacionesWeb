Multimedia en HTML: Audio y Video

1. Inserción de Audio con <audio>

El elemento <audio> permite incorporar sonido directamente en páginas web sin necesidad de plugins externos.

Estructura básica

<audio src="ruta-al-archivo.mp3" controls></audio>

Ejemplo con controles básicos

Nota: Este es un audio de ejemplo. En un sitio real, reemplazarías la ruta con tu archivo de audio.

Múltiples formatos (compatibilidad entre navegadores)

<audio controls> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> <source src="audio.wav" type="audio/wav"> Tu navegador no soporta el elemento de audio. </audio>

Atributos principales

Atributo Descripción Valores
controls Muestra los controles de reproducción No tiene valor (presencia = true)
autoplay Reproduce automáticamente el audio al cargar No tiene valor (presencia = true)
loop Repite el audio cuando termina No tiene valor (presencia = true)
muted Inicia el audio silenciado No tiene valor (presencia = true)
preload Especifica si el audio debe precargarse auto, metadata, none

Ejemplo completo con todos los atributos

Este audio tiene configurado:

  • controls: Muestra los controles de reproducción
  • autoplay: Intenta reproducir automáticamente (muchos navegadores bloquean esto)
  • loop: Se repite al finalizar
  • muted: Comienza silenciado
  • preload="auto": Sugiere precargar el archivo completo

Importante: Los navegadores modernos bloquean la reproducción automática (autoplay) de archivos con sonido. Para que el autoplay funcione, generalmente debe combinarse con el atributo muted.

2. Inserción de Video con <video>

El elemento <video> permite incorporar videos directamente en páginas web sin necesidad de plugins externos.

Estructura básica

<video src="ruta-al-archivo.mp4" controls></video>

Ejemplo con controles básicos

Nota: Este es un video de ejemplo. En un sitio real, reemplazarías la ruta con tu archivo de video.

Múltiples formatos (compatibilidad entre navegadores)

<video controls width="640" height="360"> <source src="video.mp4" type="video/mp4"> <source src="video.webm" type="video/webm"> <source src="video.ogg" type="video/ogg"> Tu navegador no soporta el elemento de video. </video>

Atributos principales

Atributo Descripción Valores
controls Muestra los controles de reproducción No tiene valor (presencia = true)
autoplay Reproduce automáticamente el video al cargar No tiene valor (presencia = true)
loop Repite el video cuando termina No tiene valor (presencia = true)
muted Inicia el video silenciado No tiene valor (presencia = true)
poster Imagen a mostrar antes de reproducir el video URL de la imagen
preload Especifica si el video debe precargarse auto, metadata, none
width Ancho del video en píxeles Valor numérico
height Alto del video en píxeles Valor numérico
playsinline Reproduce en línea (no pantalla completa) en iOS No tiene valor (presencia = true)

Ejemplo completo con todos los atributos

Este video tiene configurado:

  • controls: Muestra los controles de reproducción
  • autoplay: Intenta reproducir automáticamente (muchos navegadores bloquean esto)
  • loop: Se repite al finalizar
  • muted: Comienza silenciado
  • poster: Muestra una imagen antes de reproducirse
  • width/height: Establece dimensiones específicas
  • playsinline: Reproduce dentro de la página en dispositivos iOS
  • preload="auto": Sugiere precargar el video completo

3. Formatos Compatibles y Consideraciones

Formatos de Audio Comunes

Formato Extensión MIME Type Compatibilidad
MP3 .mp3 audio/mpeg Excelente (todos los navegadores modernos)
WAV .wav audio/wav Buena (mayor tamaño de archivo)
OGG .ogg audio/ogg Buena (no en Safari)
AAC .aac, .m4a audio/aac, audio/mp4 Buena (especialmente en Safari/iOS)

Formatos de Video Comunes

Formato Extensión MIME Type Compatibilidad
MP4 (H.264) .mp4 video/mp4 Excelente (todos los navegadores modernos)
WebM .webm video/webm Muy buena (no en Safari antiguo)
OGG Theora .ogv video/ogg Limitada (principalmente Firefox)

Consideraciones importantes

Accesibilidad

Para mejorar la accesibilidad, considere añadir:

  • Subtítulos con el elemento <track>:
<video controls> <source src="video.mp4" type="video/mp4"> <track src="subtitulos.vtt" kind="subtitles" srclang="es" label="Español"> </video>

Consideraciones de rendimiento

  • Tamaño de archivo: Los archivos multimedia pueden ser grandes. Optimiza tus archivos para web.
  • Ancho de banda: Considera ofrecer diferentes calidades según la conexión del usuario.
  • Reproductor alternativo: Proporciona una alternativa para navegadores antiguos.
  • Preload: Usa preload="none" o preload="metadata" para páginas donde el multimedia no es el contenido principal.

Estrategias para autoplay

Para conseguir que el autoplay funcione en navegadores modernos:

  • Combinar autoplay con muted
  • Iniciar la reproducción como respuesta a una interacción del usuario
  • Utilizar JavaScript para gestionar la reproducción
// Ejemplo de inicio con JavaScript tras interacción document.getElementById('boton-reproducir').addEventListener('click', function() { document.getElementById('mi-video').play(); });
