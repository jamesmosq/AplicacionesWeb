Tipos de listas en HTML

Listas no ordenadas

Se utilizan cuando el orden de los elementos no es importante:

  • Característica principal: elementos precedidos por viñetas
  • Etiqueta principal: <ul>
  • Cada elemento se define con: <li>

Listas ordenadas

Se utilizan cuando el orden de los elementos es importante:

  1. Paso 1: Planificar la estructura
  2. Paso 2: Escribir el HTML
  3. Paso 3: Aplicar estilos con CSS
  4. Paso 4: Añadir interactividad con JavaScript

Listas de definición

Se utilizan para crear listas de términos y sus definiciones:

HTML
HyperText Markup Language - Lenguaje estándar para crear páginas web.
CSS
Cascading Style Sheets - Lenguaje utilizado para describir la presentación de documentos HTML.
JavaScript
Lenguaje de programación que permite crear contenido dinámico en páginas web.

  • W3Schools - Tutoriales y referencias sobre tecnologías web

Enlaces internos

Los enlaces internos permiten la navegación dentro de la misma página:

  • Ir a la sección de listas
  • Ir a los ejercicios prácticos

Atributos importantes

href
Especifica la URL de destino del enlace.
target
Define dónde se abrirá el enlace (ej. _blank para una nueva pestaña).
rel
Especifica la relación entre la página actual y la página enlazada.
title
Proporciona información adicional que aparece como tooltip.
