Tipos de listas en HTML

Lista no ordenada (ul)

Se usa cuando el orden de los elementos no es importante.

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4

Lista no ordenada con estilo personalizado

  • Elemento con viñeta cuadrada
  • Otro elemento cuadrado
  • Tercer elemento cuadrado

Lista ordenada (ol)

Se usa cuando el orden de los elementos es importante.

  1. Primer paso
  2. Segundo paso
  3. Tercer paso
  4. Cuarto paso

Lista ordenada con tipo personalizado

  1. Elemento A
  2. Elemento B
  3. Elemento C

Lista ordenada con inicio personalizado

  1. Este elemento comenzará en 10
  2. Este será el 11
  3. Este será el 12

Lista ordenada con tipo romano

  1. Elemento I
  2. Elemento II
  3. Elemento III

Listas anidadas

Puedes incluir listas dentro de otras listas.

  • Frutas
    • Manzanas
    • Naranjas
    • Plátanos
  • Verduras
    • Zanahorias
    • Brócoli
    • Espinacas

Lista de definición (dl)

Se usa para crear listas de términos y sus definiciones.

HTML
HyperText Markup Language - Lenguaje estándar para crear páginas web.
CSS
Cascading Style Sheets - Lenguaje utilizado para describir la presentación de documentos HTML.
JavaScript
Lenguaje de programación que permite crear contenido dinámico en páginas web.
