Atributos avanzados de enlaces en HTML

Tabla de atributos de enlaces

Atributo Descripción Valores comunes
href La URL a la que apunta el enlace URL, ruta relativa, #id-interno
target Dónde abrir el enlace _blank, _self, _parent, _top
rel La relación entre la página actual y la página enlazada nofollow, noreferrer, noopener, external
title Información adicional sobre el enlace (aparece como tooltip) Texto descriptivo
download Indica que el recurso debe descargarse Nombre de archivo o vacío
hreflang El idioma del documento enlazado es, en, fr, etc.
type El tipo MIME del recurso enlazado text/html, image/jpeg, application/pdf

Ejemplos prácticos

Target: Controlar dónde se abre el enlace

Abrir en nueva pestaña (_blank)

Abrir en la misma pestaña (_self)

Abrir en el marco padre (_parent)

Abrir en la ventana completa (_top)

Nota: Los valores _parent y _top son útiles cuando se trabaja con marcos (frames) o iframes.

Rel: Definir relaciones

Nofollow: Indica a los buscadores que no sigan este enlace

Noreferrer: No envía información de referencia

Noopener: Previene que la nueva página acceda a window.opener

External: Indica que es un enlace externo

Buenas prácticas

  • Siempre usa rel="noopener noreferrer" con target="_blank" por seguridad
  • Usa títulos descriptivos que aporten información adicional, no repitas el texto del enlace
  • Asegúrate de que el texto del enlace sea descriptivo (evita "haz clic aquí" o "más")
  • Usa rel="nofollow" para enlaces que no quieras avalar o para enlaces pagados
