Enlaces internos

Los enlaces internos te permiten navegar dentro de la misma página.

Para crear un enlace interno, primero debes crear un elemento con un atributo "id", y luego hacer referencia a ese id con un "#" en el atributo href.

Atributos de enlaces

Atributo target

El atributo "target" especifica dónde abrir el enlace.

  • Abrir GitHub en una nueva pestaña (_blank)
  • Abrir Mozilla en la misma pestaña (_self, es el valor predeterminado)

Atributo title

El atributo "title" muestra un texto informativo cuando el usuario pasa el cursor sobre el enlace.

Pasar el cursor sobre este enlace para ver el título

Atributo rel

El atributo "rel" especifica la relación entre la página actual y la página enlazada.

  • Enlace con rel="nofollow" (indica a los motores de búsqueda que no sigan este enlace)
  • Enlace con rel="noreferrer" (no envía información de referencia)
  • Enlace con rel="noopener" (previene que la nueva página acceda a la ventana original)
