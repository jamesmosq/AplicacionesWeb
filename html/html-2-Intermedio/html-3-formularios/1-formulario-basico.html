Formularios HTML - Estructura Básica

Los formularios HTML permiten recopilar información de los usuarios. La estructura básica de un formulario se define con la etiqueta <form>.

Atributos importantes de la etiqueta <form>:

  • action: URL donde se enviarán los datos del formulario
  • method: Método HTTP utilizado para enviar los datos (GET o POST)
  • name: Nombre del formulario para referenciarlo con JavaScript

Ejemplo de Formulario Básico

<form action="/procesar.php" method="post" name="formularioContacto">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    
    <label for="email">Correo electrónico:</label>
    <input type="email" id="email" name="email">
    
    <label for="mensaje">Mensaje:</label>
    <input type="text" id="mensaje" name="mensaje">
    
    <button type="submit">Enviar</button>
</form>

Elementos clave en un formulario:

  • Etiquetas (label): Describen el propósito de cada campo
  • Campos de entrada (input): Permiten al usuario ingresar datos
  • Botones: Permiten enviar o resetear el formulario

Es una buena práctica siempre asociar una etiqueta <label> con su campo correspondiente usando el atributo for que coincide con el id del campo.

Estructura de un formulario más completo

Información Personal
Información de Contacto 
<form action="/procesar-avanzado.php" method="post">
    <fieldset>
        <legend>Información Personal</legend>
        
        <label for="nombre2">Nombre:</label>
        <input type="text" id="nombre2" name="nombre">
        
        <label for="apellido">Apellido:</label>
        <input type="text" id="apellido" name="apellido">
    </fieldset>
    
    <fieldset>
        <legend>Información de Contacto</legend>
        
        <label for="email2">Correo electrónico:</label>
        <input type="email" id="email2" name="email">
        
        <label for="telefono">Teléfono:</label>
        <input type="tel" id="telefono" name="telefono">
    </fieldset>
    
    <button type="submit">Enviar</button>
    <button type="reset">Limpiar</button>
</form>

Organización con fieldset y legend:

  • <fieldset>: Agrupa campos relacionados
  • <legend>: Proporciona un título al grupo fieldset

Esto mejora la organización y accesibilidad del formulario, especialmente para formularios más grandes.
