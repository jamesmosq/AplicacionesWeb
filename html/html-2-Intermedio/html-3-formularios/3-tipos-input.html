Formularios HTML - Tipos de Input

HTML5 introdujo varios tipos de <input> que mejoran la experiencia de usuario y añaden validaciones automáticas.

Estos tipos de input proporcionan interfaces especializadas en dispositivos móviles y navegadores modernos.

Tipo Descripción Ejemplo
text Campo de texto de una sola línea (predeterminado) <input type="text">
password Campo de contraseña que oculta los caracteres <input type="password">
email Campo para direcciones de correo electrónico <input type="email">
number Campo para valores numéricos <input type="number">
tel Campo para números telefónicos <input type="tel">
url Campo para URLs <input type="url">
date Selector de fecha <input type="date">
