Formularios HTML - Elementos de Entrada

HTML ofrece varios elementos para capturar diferentes tipos de datos de los usuarios:

  • <input>: El más versátil, con muchos tipos diferentes
  • <textarea>: Para texto multilínea
  • <select>: Para listas desplegables y opciones
  • <button>: Para acciones como enviar o reiniciar el formulario

1. El elemento input

El elemento <input> es el más común y versátil. Su comportamiento cambia según el atributo type.

<input type="text" id="nombre_input" name="nombre" placeholder="Escribe tu nombre">
<input type="email" id="email_input" name="email" placeholder="ejemplo@dominio.com">
<input type="password" id="password_input" name="password" placeholder="Tu contraseña">
<input type="number" id="numero_input" name="numero" min="0" max="100" step="1">
<input type="date" id="fecha_input" name="fecha">
<input type="color" id="color_input" name="color">
<input type="range" id="rango_input" name="rango" min="0" max="100">

Checkbox (type="checkbox"):

<input type="checkbox" id="check1" name="intereses" value="deportes">
<label for="check1">Deportes</label>

<input type="checkbox" id="check2" name="intereses" value="musica">
<label for="check2">Música</label>

<input type="checkbox" id="check3" name="intereses" value="lectura">
<label for="check3">Lectura</label>

Radio (type="radio"):

<input type="radio" id="radio1" name="genero" value="masculino">
<label for="radio1">Masculino</label>

<input type="radio" id="radio2" name="genero" value="femenino">
<label for="radio2">Femenino</label>

<input type="radio" id="radio3" name="genero" value="otro">
<label for="radio3">Otro</label>
<input type="file" id="archivo_input" name="archivo">

2. El elemento textarea

El elemento <textarea> permite a los usuarios ingresar texto multilínea.

<textarea id="comentario" name="comentario" rows="5" placeholder="Escribe tu comentario aquí..."></textarea>

Atributos importantes de <textarea>:

  • rows: Número de líneas visibles
  • cols: Número de caracteres visibles por línea
  • placeholder: Texto de ejemplo
  • maxlength: Número máximo de caracteres permitidos

3. El elemento select

El elemento <select> crea una lista desplegable de opciones.

<select id="pais" name="pais">
    <option value="">Selecciona un país</option>
    <option value="es">España</option>
    <option value="mx">México</option>
    <option value="ar">Argentina</option>
    <option value="co">Colombia</option>
    <option value="pe">Perú</option>
</select>

Selección múltiple:

Mantén presionada la tecla Ctrl (Cmd en Mac) para seleccionar múltiples opciones

<select id="lenguajes" name="lenguajes[]" multiple size="4">
    <option value="html">HTML</option>
    <option value="css">CSS</option>
    <option value="js">JavaScript</option>
    <option value="php">PHP</option>
    <option value="python">Python</option>
    <option value="java">Java</option>
</select>

Grupos de opciones:

<select id="vehiculos" name="vehiculo">
    <optgroup label="Coches">
        <option value="sedan">Sedán</option>
        <option value="suv">SUV</option>
        <option value="deportivo">Deportivo</option>
    </optgroup>
    <optgroup label="Motocicletas">
        <option value="scooter">Scooter</option>
        <option value="deportiva">Deportiva</option>
        <option value="touring">Touring</option>
    </optgroup>
</select>
