Atributos esenciales de imágenes en HTML

La etiqueta <img> puede tener varios atributos para controlar cómo se muestra la imagen y proporcionar información adicional.

Atributo Descripción Obligatorio
src URL o ruta de la imagen Sí
alt Texto alternativo para la imagen Sí (para accesibilidad)
width Ancho de la imagen en píxeles No
height Alto de la imagen en píxeles No
title Texto que aparece al pasar el cursor No
loading Cómo debe cargarse la imagen ("lazy" o "eager") No

src (source)

Este atributo especifica la URL o ruta a la imagen que deseas mostrar.

<img src="https://via.placeholder.com/300x150" alt="Ejemplo de atributo src">

Importante: El atributo src es obligatorio para la etiqueta img.

alt (texto alternativo)

Este atributo proporciona un texto alternativo que se muestra si la imagen no puede cargarse o para usuarios que utilizan lectores de pantalla.

<img src="imagen_que_no_existe.jpg" alt="Esta imagen no existe, pero puedes ver este texto alternativo">

Importante: El atributo alt es fundamental para:
  • Accesibilidad: permite a personas con discapacidad visual entender el contenido
  • SEO: ayuda a los motores de búsqueda a indexar tu contenido
  • Mostrar información cuando las imágenes fallan

width y height (ancho y alto)

Estos atributos especifican las dimensiones de la imagen en píxeles.

Imagen con dimensiones originales:

<img src="https://via.placeholder.com/300x150" alt="Imagen original">

Imagen con width y height personalizados:

<img src="https://via.placeholder.com/300x150" alt="Imagen redimensionada" width="200" height="100">

Buena práctica: Siempre especifica el width y height de tus imágenes para:
  • Reservar espacio en la página mientras las imágenes se cargan
  • Evitar el "layout shift" (cambio de diseño) cuando la página se carga
  • Mejorar el rendimiento de la página

title (título)

Este atributo muestra un texto informativo cuando el usuario pasa el cursor sobre la imagen.

<img src="https://via.placeholder.com/300x150" alt="Imagen con título" title="Este texto aparece al pasar el cursor sobre la imagen">

Pasa el cursor sobre la imagen de arriba para ver el título.

loading (carga)

Este atributo controla cómo el navegador debe cargar la imagen.

Carga normal (eager):

<img src="https://via.placeholder.com/300x150" alt="Carga normal" loading="eager">

Carga diferida (lazy):

<img src="https://via.placeholder.com/300x150" alt="Carga diferida" loading="lazy">

Ventaja: La carga diferida (lazy loading) mejora el rendimiento de la página al cargar las imágenes solo cuando están a punto de entrar en el viewport.
