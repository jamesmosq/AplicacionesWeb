Inserción de imágenes en HTML

La etiqueta <img> se utiliza para insertar imágenes en una página web. Es una etiqueta de "auto-cierre", lo que significa que no necesita una etiqueta de cierre.

Sintaxis básica

<img src="ruta/a/la/imagen.jpg" alt="Texto alternativo">

Ejemplo práctico:

Código utilizado:

<img src="https://via.placeholder.com/300x200" alt="Imagen de ejemplo 300x200">

Rutas de imágenes

1. Rutas absolutas (URLs completas)

<img src="https://via.placeholder.com/200x100/ff0000/ffffff" alt="Imagen con URL absoluta">

2. Rutas relativas (desde la raíz del sitio)

<img src="/images/logo.png" alt="Logo del sitio">

3. Rutas relativas (desde el directorio actual)

<img src="images/foto.jpg" alt="Fotografía">

4. Rutas relativas (directorio superior)

<img src="../recursos/banner.jpg" alt="Banner">

Nota: Para los ejemplos, estamos usando imágenes de placeholder.com ya que es un servicio que genera imágenes de muestra. En un sitio web real, utilizarías tus propias imágenes almacenadas en tu servidor o servicio de alojamiento.
