Tablas HTML - Atributos colspan y rowspan

Los atributos colspan y rowspan permiten que una celda ocupe más de una columna o fila:

  • colspan="n": Hace que una celda ocupe "n" columnas
  • rowspan="n": Hace que una celda ocupe "n" filas

Ejemplo de colspan

Información de Ventas Trimestrales
Producto Vendedor Cantidad
Laptops Juan 15
Smartphones Laura 25

Código utilizado:

<table>
    <tr>
        <th colspan="3">Información de Ventas Trimestrales</th>
    </tr>
    <tr>
        <th>Producto</th>
        <th>Vendedor</th>
        <th>Cantidad</th>
    </tr>
    <tr>
        <td>Laptops</td>
        <td>Juan</td>
        <td>15</td>
    </tr>
    <tr>
        <td>Smartphones</td>
        <td>Laura</td>
        <td>25</td>
    </tr>
</table>

Ejemplo de rowspan

Categoría Producto Precio
Electrónica Laptop $1200
Tablet $400
Hogar Lavadora $600
Refrigerador $950

Código utilizado:

<table>
    <tr>
        <th>Categoría</th>
        <th>Producto</th>
        <th>Precio</th>
    </tr>
    <tr>
        <td rowspan="2">Electrónica</td>
        <td>Laptop</td>
        <td>$1200</td>
    </tr>
    <tr>
        <td>Tablet</td>
        <td>$400</td>
    </tr>
    <tr>
        <td rowspan="2">Hogar</td>
        <td>Lavadora</td>
        <td>$600</td>
    </tr>
    <tr>
        <td>Refrigerador</td>
        <td>$950</td>
    </tr>
</table>

Ejemplo combinando colspan y rowspan

Información Primer Semestre
Q1 Q2
Región Norte Producto A 500 650
Producto B 300 480

Código utilizado:

<table>
    <tr>
        <th colspan="2" rowspan="2">Información</th>
        <th colspan="2">Primer Semestre</th>
    </tr>
    <tr>
        <th>Q1</th>
        <th>Q2</th>
    </tr>
    <tr>
        <td rowspan="2">Región Norte</td>
        <td>Producto A</td>
        <td>500</td>
        <td>650</td>
    </tr>
    <tr>
        <td>Producto B</td>
        <td>300</td>
        <td>480</td>
    </tr>
</table>
