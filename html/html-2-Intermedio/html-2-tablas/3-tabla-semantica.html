Tablas HTML - Elementos Semánticos

Las tablas HTML pueden utilizar elementos semánticos para mejorar la estructura y el significado:

  • <caption>: Define un título o leyenda para la tabla
  • <thead>: Agrupa el contenido de la cabecera de la tabla
  • <tbody>: Agrupa el cuerpo principal de la tabla
  • <tfoot>: Agrupa el contenido del pie de la tabla

Ejemplo de tabla con elementos semánticos

Ventas Trimestrales por Departamento - 2024
Departamento Q1 Q2 Q3 Q4
Electrónica $45,000 $58,200 $52,300 $78,500
Ropa $32,600 $35,400 $42,800 $68,200
Hogar $28,300 $31,500 $36,700 $49,800
Total $105,900 $125,100 $131,800 $196,500

Código utilizado:

<table>
    <caption>Ventas Trimestrales por Departamento - 2024</caption>
    <thead>
        <tr>
            <th>Departamento</th>
            <th>Q1</th>
            <th>Q2</th>
            <th>Q3</th>
            <th>Q4</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Electrónica</td>
            <td>$45,000</td>
            <td>$58,200</td>
            <td>$52,300</td>
            <td>$78,500</td>
        </tr>
        <tr>
            <td>Ropa</td>
            <td>$32,600</td>
            <td>$35,400</td>
            <td>$42,800</td>
            <td>$68,200</td>
        </tr>
        <tr>
            <td>Hogar</td>
            <td>$28,300</td>
            <td>$31,500</td>
            <td>$36,700</td>
            <td>$49,800</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Total</td>
            <td>$105,900</td>
            <td>$125,100</td>
            <td>$131,800</td>
            <td>$196,500</td>
        </tr>
    </tfoot>
</table>

Beneficios de usar elementos semánticos:

  • Mejora la accesibilidad para lectores de pantalla
  • Facilita la aplicación de estilos CSS específicos
  • Hace que el código HTML sea más organizado y comprensible
  • La etiqueta <tfoot> se mostrará al final visualmente, pero puede aparecer antes de <tbody> en el código
