Tablas HTML - Estilizado Básico

El estilizado de tablas HTML con CSS permite mejorar la presentación y usabilidad de los datos. Veremos distintas técnicas para estilizar tablas:

  • Bordes y espaciado
  • Colores de fondo para encabezados y filas
  • Filas con colores alternados (efecto cebra)
  • Efectos de hover
  • Tablas responsivas

Para el efecto cebra usamos el selector tr:nth-child(even) que permite aplicar estilos a las filas pares, creando el efecto de alternancia de colores.

.tabla-zebra tr:nth-child(even) {
    background-color: #f2f2f2;
}

El efecto hover resalta la fila sobre la que pasa el cursor, mejorando la usabilidad de la tabla. Se implementa con:

.tabla-hover tr:hover {
    background-color: #f5f5f5;
    transition: background-color 0.3s;
}

La propiedad transition añade una animación suave al cambio de color.
